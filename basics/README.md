# Python Basics

This folder contains resources and notes on the basics of Python programming. It covers fundamental concepts, syntax, and commonly used tools that are essential for beginners to get started with Python. Whether you're new to programming or just new to Python, this folder will help you build a strong foundation in the language.

## File Guide

- `1_fundamentals.py` - Introduces basic Python syntax, comments, print statements, variables, input/output, conditionals, loops, and simple functions.
- `2_classes.py` - Covers object-oriented programming basics, including classes, objects, constructors, attributes, methods, class vs. instance state, and inheritance.
- `3_data_structures.py` - Explains Python core data structures: lists, tuples, dictionaries, sets, and comprehensions for lists, dicts, and sets.
- `4_dunder_and_protocols.py` - Demonstrates object representation with `__str__`/`__repr__`, the iterator protocol, context managers, and operator overloading.
- `5_memory_and_internals.py` - Covers mutable vs. immutable types, identity vs. equality, variable scope layers, and basic garbage collection concepts.
- `6_functional_and_generators.py` - Introduces first-class functions, closures, decorators, and generator functions.
- `7_exceptions_and_modules.py` - Explains exception bubbling and handling, import resolution, absolute vs. relative imports, and the script entrypoint with `if __name__ == '__main__'`.
- `8_builtin_functions.py` - Shows common Python built-ins like lambda, map, filter, sum, max, min, any, all, zip, enumerate, reversed, sorted, attribute helpers, and more.

## How to Use This Folder

Each file is written as a self-contained tutorial script. Run individual files to see examples and explanations in action.

## Note

These files are designed for learning and experimentation, so the code is intentionally simple and explanatory.
